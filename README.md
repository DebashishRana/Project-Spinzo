## Differentiation

We engineered Spinzo with specific AI features designed to showcase **Visible Machine Cognition**.

### 1. 🪤 The "Akinator Trap" (Contradiction Resolution)
If a user answers "Yes" to *Is the player a fast bowler?* and later says "Yes" to *Is the player a spinner?*, Spinzo doesn't break. The **Contradiction Agent** intercepts the conflict, flags it on the UI, and mathematically recalibrates its certainty.

### 2. 🧠 The "Mind Reader" Cognitive Dashboard
We don't hide the math. Next to the chat, a live dashboard shows:
*   **Top 3 Player Probabilities** adjusting in real-time (e.g., *Kohli: 45% ↗ 82%*).
*   **Why this question?** (e.g., *"Asking about captaincy to mathematically eliminate 68 non-captains."*)

### 3. 🎭 "Vibe Coding" via Gemini (Dynamic Persona)
Spinzo's personality changes based on Bayesian state:
*   **High Uncertainty (Flat probabilities):** *"This is tricky. You've given me conflicting clues. Let's try a different angle..."*
*   **High Confidence (>80% for one player):** *"I think I have it. Just to be absolutely sure..."*

### 4. 📊 Post-Game Intelligence Report
When the game ends, Spinzo generates a detailed breakdown of its thought process, highlighting the **"Pivot Question"**—the exact moment it figured you out. *(e.g., "When you said YES to 'Left-arm spinner', my confidence in Jadeja spiked from 12% to 89%.")*

### 5. 🏢 Beyond Cricket: The Enterprise B2B Angle
While framed as a game, Spinzo's core architecture is an **Entropy-Driven Diagnostic Engine**. This exact mathematical framework scales directly to:
*   **Medical Triage:** Narrowing down patient diagnoses with the fewest questions.
*   **IT Helpdesk:** Troubleshooting server errors autonomously.
*   **E-Commerce:** Finding the perfect product for a user instantly.

## 🚀 Run It in 3 Steps

**1. Clone & Install**
```bash
git clone https://github.com/your-username/spinzo.git
cd spinzo
```

**2. Start the Backend (FastAPI)**
```bash
cd backend
pip install -r requirements.txt
# Add your GEMINI_API_KEY to a .env file
uvicorn main:app --reload
```

**3. Start the Frontend (React)**
```bash
cd ../frontend
npm install
npm run dev
```
